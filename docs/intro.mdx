# My Interview Vault

Welcome to the vault. This site is a docs-as-code interview preparation hub:
every guide lives as a Markdown or MDX file in the repository.

## How it works

- Read the guides from the website.
- Improve a page by editing the matching file in `docs/`.
- Open a pull request on GitHub.
- Merged changes publish to the live site.

## Content areas

- Data Structures: complexity cheat sheets, bit manipulation hacks, graphs, and Blind Top 75 walkthroughs.
- System design: scalable architecture patterns, real-world case studies, concepts, and tradeoff matrices.
- Behavioral prep: STAR stories, leadership principles, conflict resolution, and behavioral frameworks.
- AI/ML: machine learning systems design, core ML/DL concepts, and roadmaps.
- TPM: Technical Program Management metrics, execution strategies, and system delivery.
- Interview and Beyond: mock platforms, company prep guides (Google, Meta, OpenAI), and career tips.

## First pass

Start with one high-signal page per category. The vault should prefer clear,
reviewable notes over bulk collections of links.
